Introdução

Boas vindas!

É muito importante proporcionar uma experiência livre de assédio para todas as pessoas participantes. Por favor, leia o Código de Conduta da R-Ladies.

O R Markdown é um pacote no R que possibilita criar relatórios reprodutíveis. Com este tipo de arquivo (.Rmd), é possível adicionar textos, códigos, os resultados dos códigos, e muito mais!

Ilustração por Allison Horst.

Se você é uma pessoa que utiliza R, conheceu as possibilidades de utilizar R Markdown (e os pacotes que expandem mais ainda as possibilidades), e gostaria de começar a utilizá-lo, a minha dica é:

Conheça Markdown

Porque? O R Markdown utiliza a linguagem de marcação Markdown. O Markdown também é usado em outros lugares, como:

  • arquivos README.md no GitHub (exemplo aqui!)

  • no site Hackmd.io (muito útil para escrever colaborativamente)

  • em fóruns (como o Discourse da Curso-R) onde você pode marcar o texto da sua postagem

  • entre muitos outros!

Se você tem interesse em aprender Markdown, esse tutorial prático é para você!

Ênfase

Negrito

Para destacar um texto em negrito, coloque ** ou __ ao redor do texto.

Por exemplo:

Esse é um texto com uma palavra destacada em negrito.

Esse é um texto com uma palavra destacada em **negrito**.

Esse é um texto com uma palavra destacada em __negrito__.

Esse é um texto com uma palavra destacada em negrito.

Itálico

Para destacar um texto em itálico, coloque * ou _ ao redor do texto.

Por exemplo:

Esse é um texto com uma palavra destacada em itálico.

Esse é um texto com uma palavra destacada em *itálico*.

Esse é um texto com uma palavra destacada em _itálico_.

Esse é um texto com uma palavra destacada em itálico.

Riscado

Para riscar um texto, coloque ~~ ao redor do texto.

Por exemplo:

Esse é um texto com uma palavra riscada.

Esse é um texto com uma palavra ~~riscada~~.

Esse é um texto com uma palavra riscada.

Títulos

Os títulos funcionam como uma hierarquia, e para criar um título é necessário colocar um # no início da linha. Então um # marca um título, ## marca um sub-título, e assim sucessivamente. Veja os exemplos 😉

# Título 1

Título 1

## Título 2

Título 2

### Título 3

Título 3

Listas

Listas ordenadas

Você pode fazer uma lista ordenada usando somente números. Você pode repetir o número quantas vezes quiser:

1. Maçã

1. Banana

1. Uva

O resultado será:

  1. Maçã
  2. Banana
  3. Uva

Listas não ordenadas

Você pode fazer uma lista não ordenada escrevendo com hifens ou asteriscos, como a seguir:

* Maçã

* Banana

* Uva

- Maçã

- Banana

- Uva

O resultado será:

  • Maçã
  • Banana
  • Uva

Você também pode adicionar sub-itens na lista:

  • Frutas
    • Maçã
    • Banana
    • Uva

Links

  • Você pode criar um link utilizando essa estrutura: [texto](http://url-da-pagina.com).

Por exemplo:

Você pode consultar mais [materiais sobre R nesta página](https://beatrizmilz.github.io/slidesR/).

Você pode consultar mais materiais sobre R nesta página.

Imagens

  • Você pode incluir arquivo de imagem utilizando essa estrutura: ![descricao da imagem](http://url-da-imagem.com). A descrição da imagem é importante para acessibilidade do conteúdo através de leitores de tela.
